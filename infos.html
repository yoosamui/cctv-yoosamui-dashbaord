🎯 Basic Filters
Camera Name
Filter footage from a specific camera. Works with any combination of uppercase/lowercase letters.
Example: Gate or garage or ENTRANCE
today
Show all footage from the current day (midnight to present).
Example: today or Gate today
YYYY-MM-DD
Show all footage from a specific date.
Example: 2026-06-13 or Gate 2026-06-13
HH:MM Time Filter
Show footage from the most recent date at or after the specified time.
Example: 17:00 or Gate 17:00
🔗 Combined Filters
Camera + Date
Filter footage from a specific camera on a specific date.
Example: Gate 2026-06-13 or garage 2026-06-12
Camera + Time
Filter footage from a specific camera from the specified time onward on the most recent date.
Example: Gate 18:30 or garage 06:00
Camera + Date + Time
Filter footage from a specific camera on a specific date from the specified time onward.
Example: Gate 2026-06-13 17:00
Date + Time
Filter footage from all cameras on a specific date from the specified time onward.
Example: 2026-06-13 17:00
⚡ Flexible Date/Time Formats
YYYY-MM-DD HH
Hour only (minutes default to 00).
Example: 2026-06-13 17 or Gate 2026-06-13 17
YYYY-MM-DD:HH:MM
Colon separator between date and time.
Example: 2026-06-13:17:30 or Gate 2026-06-13:17:30
YYYY-MM-DD:HH
Colon separator with hour only.
Example: 2026-06-13:17 or Gate 2026-06-13:17
YYYY-MM-DD-HH:MM
Dash separator between date and time.
Example: 2026-06-13-17:30 or Gate 2026-06-13-17:30

📝 Text Search

Any text that doesn't match the above formats will be treated as a text search, looking for matches in filenames and paths.

person → Shows all files with "person" in the name
Gate motion → Shows Gate files with "motion" in the name

🎬 Media Type Filter

After searching, use the dropdown to filter results by media type:

  • 📹 All media - Videos + Images
  • 🎥 Videos only - Only video files (.mp4, .mov, etc.)
  • 🖼️ Images only - Only detection images (.jpg, .png, etc.)

📊 Sort Order

  • 📅 Oldest First - Chronological order (earliest to latest)
  • 🖼️ Detection Images - Maximum 3 images per video
  • 📁 Grouped by Video - Images stay attached to their source video

🔍 Quick Examples

  • gate → All Gate footage
  • garage today → Garage footage from today
  • entrance 2026-06-13 → Entrance footage from June 13
  • behind 22:00 → Behind footage from 10 PM onward
  • 2026-06-13:17 → All cameras from June 13 at 5 PM

🌐 API Endpoints

GET /api/footage?query=<query> - Search footage
GET /api/preview?path=<path> - Preview a file
GET /api/version - Get API version
